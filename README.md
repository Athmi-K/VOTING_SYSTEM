# 🗳️ Voting System

A secure online voting system developed using the MERN stack.

## 📌 Features

- Student Registration
- Login Authentication
- OTP Verification
- Candidate Management
- Secure Vote Casting
- Election Results
- Admin Dashboard

## 🛠️ Tech Stack

### Frontend
- React
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

## 📂 Project Structure

```
Voting_System/
│
├── frontend/

├── README.md

```

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/VOTING_SYSTEM.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
